## 自定义语法
### 从者放技能
`s:1,2,3` 表示一号从者放 2 技能（单体技能给三号位从者）

> 宇宙凛的色卡选择同理，红 1 蓝 2 绿 3

### 御主技能
`m:1,0,1` 表示放衣服 1 技能（第二位是占位的别睬他）给一号位从者

### 换人服
`c:1,5,0` 代表一号位从者和五号位从者交换
